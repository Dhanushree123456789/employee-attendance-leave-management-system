====================================================================

STEP 1: EXTRACT FILES
----------------------
Unzip the attendance-system.zip file to your desired location

STEP 2: INSTALL PYTHON
-----------------------
Ensure Python 3.8+ is installed on your system
Download from: https://www.python.org/downloads/

STEP 3: SETUP ENVIRONMENT
--------------------------
Open terminal/command prompt in the project directory:

# Create virtual environment (recommended)
python -m venv venv

# Activate virtual environment
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

STEP 4: INSTALL DEPENDENCIES
-----------------------------
pip install -r requirements.txt

This installs:
- Flask 2.3.3 (Web framework)
- Werkzeug 2.3.7 (WSGI utilities)

STEP 5: INITIALIZE DATABASE
----------------------------
python database/init_db.py

This creates the SQLite database with:
‚úì 4 tables (roles, users, attendance, leave_requests)
‚úì Default admin account
‚úì 3 sample employee accounts

STEP 6: RUN APPLICATION
------------------------
python app.py

The server will start on: http://localhost:5000

STEP 7: ACCESS SYSTEM
----------------------
Open your browser and go to: http://localhost:5000

üóÑÔ∏è DATABASE SCHEMA:
====================================================================

TABLE: roles
------------
- role_id (PK)
- role_name (UNIQUE)
- description

TABLE: users
------------
- user_id (PK)
- username (UNIQUE)
- password (SHA-256 hashed)
- full_name
- email (UNIQUE)
- role_id (FK -> roles)
- date_joined
- is_active

TABLE: attendance
-----------------
- attendance_id (PK)
- user_id (FK -> users)
- attendance_date
- status (Present/Absent)
- marked_at (timestamp)
- remarks
- UNIQUE(user_id, attendance_date)

TABLE: leave_requests
---------------------
- leave_id (PK)
- user_id (FK -> users)
- leave_type (Sick/Casual/Annual)
- start_date
- end_date
- reason
- status (Pending/Approved/Rejected)
- applied_at (timestamp)
- reviewed_by (FK -> users)
- reviewed_at (timestamp)
- admin_remarks

üîß TROUBLESHOOTING:
====================================================================

ISSUE: Cannot access http://localhost:5000
SOLUTION: 
- Check if Python is installed: python --version
- Check if Flask is installed: pip list | grep Flask
- Ensure no other service is using port 5000
- Check firewall settings

ISSUE: Database not found
SOLUTION:
- Run: python database/init_db.py
- Check if database/attendance.db exists
- Verify write permissions in database folder

ISSUE: Login not working
SOLUTION:
- Use exact credentials (case-sensitive)
- Clear browser cache and cookies
- Check if database is initialized
- Verify user exists: sqlite3 database/attendance.db "SELECT * FROM users;"

ISSUE: Module not found errors
SOLUTION:
- Activate virtual environment
- Install requirements: pip install -r requirements.txt
- Check Python version: python --version (must be 3.8+)
